what is a RAID System?

RAID (Redundant Array of Independent Disks) is a way to redundently store data in a single unit that has improved performance and lower risk of permanent data loss. RAID systems can be configured in various levels, each offering different benefits and trade-offs. Common RAID levels include RAID 0 (striping), RAID 1 (mirroring), and RAID 10 (a combination of striping and mirroring). Stirpping is the proccess of spliting a file and storing it on many drives, while mirroring is the proccess of copying a file onto many drives. Both of these are solutions to hard drives most common issues, which are caused by the fact that hard drive data is stored physically. While there are industrial RAID systems that cost a LOT of money

there are much more resonably priced options for at-home use.
